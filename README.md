# BALL-POPPER
A browser-based hand gesture controlled game built with React, TypeScript, and MediaPipe Hands, allowing players to interact using real-time webcam tracking.
🎮 Hand Gesture Controlled Game

An interactive web-based game that uses real-time hand tracking to control gameplay through a webcam. Built with React, TypeScript, Vite, and Google's MediaPipe Hands, the project enables touchless interaction by detecting and tracking hand movements directly in the browser.

Features:-
✋ Real-time hand tracking
📷 Webcam-based controls
⚡ Fast and responsive gameplay
🌐 Runs entirely in the browser
🎯 Gesture-based interaction
📱 Modern React + TypeScript architecture

Tech Stack:-
1)React
2)TypeScript
3)Vite
4)MediaPipe Hands
4)HTML5 Canvas
5)Web APIs


How It Works:- 
The application accesses the user's webcam, processes video frames using MediaPipe Hands, and tracks hand landmarks in real time. These hand movements are then translated into game controls, creating a natural and immersive touch-free gaming experience.
